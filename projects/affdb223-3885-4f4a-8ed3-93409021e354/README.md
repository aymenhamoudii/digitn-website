# Eat Time — Luxury Fine Dining

This is a production-ready React + Tailwind website scaffold for "Eat Time" — a luxury fine dining restaurant. The project is designed for static hosting (Vite build output) and developer-driven updates for menu, gallery, and events.

Getting started

1. Install dependencies
   npm install

2. Development server
   npm run dev

3. Build
   npm run build

4. Preview build
   npm run preview

Content updates
- Menus, gallery images, and events are stored in JSON files under src/data. Update these files and redeploy via your usual CI/CD.

Reservations
- Reservations are phone-only. Update the phone number in src/components/ContactCard.jsx.

Accessibility & SEO
- Uses semantic HTML, visible focus indicators, and accessible components. Add page meta tags in the pages directory as needed.

Deploy
- Recommended: Netlify, Vercel, or any static hosting. Serve the contents of the dist folder.
