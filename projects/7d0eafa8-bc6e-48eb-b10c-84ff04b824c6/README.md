# Neon Jumper

A fast-paced, colorful, neon-themed infinite platformer game built purely with HTML5 Canvas, CSS3, and JavaScript. 

## Features
- Fully custom physics and AABB collision detection
- Dynamic infinite procedural platform generation
- Smooth double-jump mechanics
- Responsive neon graphics and particle effects
- Local storage for High Score tracking
- Keyboard and touch/mouse controls support

## How to Play
1. Open `index.html` in any modern web browser.
2. Click **Start Game** or press `SPACE` to begin.
3. Controls:
   - **Keyboard**: Press `SPACE`, `UP ARROW`, or `W` to jump.
   - **Mouse**: Click on the game area to jump.
   - **Touch**: Tap the screen to jump.
4. You can jump a second time while in mid-air (Double Jump).
5. Avoid falling into the gaps and hitting the sides of platforms.
6. Survive as long as possible to rack up a massive high score!

## Development details
- Built with vanilla JS and Canvas API for optimal performance.
- CSS is utilized for sleek UI overlays and neon glowing texts.
- Self-contained structure without external dependencies.
