# Restaurant Website

This restaurant website provides information about our menu, reservation options, contact details, and a gallery showcasing our dishes and ambiance. It's built using Next.js and Tailwind CSS, with a headless CMS to manage content.

## Project Structure
```
project/
├── index.html          # Main entry point
├── css/
│   └── style.css      # All styles
├── js/
│   ├── app.js         # Main JavaScript file
│   ├── contact.js      # Contact form logic
│   ├── menu.js        # Menu display logic
│   └── reservation.js   # Reservation form logic
├── components/
│   └── navbar.html    # Navigation bar
├── sections/
│   ├── contact.html    # Contact form section
│   ├── gallery.html     # Gallery section
│   ├── menu.html        # Menu section
│   └── reservation.html  # Reservation section
└── README.md           # Setup instructions
```

## Setup Instructions
1. Clone the repository or download the code.
2. Open the `index.html` file in your browser to view the website.
3. Make sure to have images available in the `images/` directory referenced in the code.
4. Modify the `menu.js`, `contact.js`, and `reservation.js` files as needed to update interactions.

## Features
- Responsive design
- Interactive menu display
- Reservations management
- Contact form for feedback
- Gallery showcasing images of dishes
