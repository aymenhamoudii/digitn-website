# Calculator

A production-ready, accessible, and responsive calculator web app.

## Features
- Fully functional calculator: add, subtract, multiply, divide
- Keyboard and mouse operation
- Polished UI with intentional color palette and fluid typography
- Accessible (WCAG 2.1 AA): semantic HTML, focus indicators, ARIA live display
- Responsive: works great on mobile, tablet, and desktop
- Secure input handling, no `eval` or unsafe DOM usage
- Clear button states and interactive feedback

## Setup & Usage
1. Download all files (`index.html`, `style.css`, `script.js`, and this `README.md`) into the same folder.
2. Open `index.html` in your web browser (Chrome, Firefox, Edge, Safari all supported).

No build steps or dependencies required.
