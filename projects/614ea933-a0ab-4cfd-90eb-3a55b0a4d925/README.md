# VIBE EATS - Premium Restaurant Web Experience

A modern, vibrant, and interactive restaurant landing page built with React and Tailwind CSS.

## Features

- **Responsive Design**: Fluid layouts optimized for mobile, tablet, and desktop.
- **Interactive Menu**: Tabbed filtering system for effortless browsing of culinary offerings.
- **Ordering System**: Real-time cart management with state tracking.
- **Booking Engine**: Functional reservation system with validation and success feedback.
- **Performance**: Lightweight animations and optimized asset handling.
- **Accessibility**: ARIA-compliant forms and semantic HTML structure.

## Tech Stack

- **React**: Component-based UI architecture.
- **Tailwind CSS**: Utility-first styling with a custom dark/vibrant theme.
- **Inter & Playfair Display**: Sophisticated typography pairing.

## Setup Instructions

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm start
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Folder Structure

- `src/components/ui`: Reusable atomic UI components.
- `src/components/sections`: Domain-specific page sections (Hero, Menu, etc.).
- `src/components/layout`: Global navigation and structural elements.
- `src/hooks`: Custom React hooks for business logic (Cart, etc.).
- `src/tests`: Comprehensive unit and logic tests.

## Design Palette
