# The Copper Fork - Restaurant Landing Page

A warm, editorial-style landing page for a fine dining restaurant featuring seasonal American cuisine.

## Features

- **Responsive Design**: Mobile-first approach with breakpoints at 768px and 1024px
- **Smooth Animations**: Scroll-triggered fade-in animations using Intersection Observer
- **Dynamic Menu**: JavaScript-powered menu rendering from seeded data
- **Reservation System**: Form handling with localStorage for demo reservations
- **Accessible**: WCAG 2.1 AA compliant with proper semantic HTML, keyboard navigation, and focus states

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Fluid Typography)
- Vanilla JavaScript (No frameworks)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. The page loads with pre-seeded demo content

No build step or server required - works directly from the file system.

## File Structure

```
├── index.html          # Main HTML document
├── css/
│   └── style.css       # Complete stylesheet
└── js/
    ├── content-data.js # Demo content seeder
    └── main.js         # Interactions & animations
```

## Design

- **Color Palette**: Deep forest green primary (#2C3E2D), copper/bronze accents (#B87333), cream backgrounds
- **Typography**: Playfair Display (headings), Source Serif 4 (body)
- **Layout**: Asymmetric grid, generous whitespace, editorial aesthetic
