TECHNOLOGIES

P2P vs GPON

Choix technologiques selon le profil utilisateur.

Point-to-Point (P2P)

  • Liaison dédiée client unique
  • Débit symétrique et garanti
  • Utilisé pour le projet SONEDE
Usage : Grands Comptes & Administrations

GPON

  • Partage via splitters optiques
  • Mutualisation des coûts infrastructure
  • Division jusqu'à 64 abonnés
Usage : Clients Résidentiels & PME
Tunisie Telecom utilise deux architectures majeures : le P2P pour des clients exigeants comme la SONEDE, et le GPON, plus économique, pour le grand public via des répartiteurs optiques.
