# The Hearth & Grain — Restaurant Page

This is a single-page static site built for local audiences with a rustic-authentic aesthetic.

Open `index.html` directly in your browser (file://) — no build steps required.

Files of interest:
- `index.html` — main document
- `css/style.css` — project styles
- `js/*.js` — interaction scripts (global scope, no modules)

Accessibility and performance notes are included in `ACCESSIBILITY.md` and `PERFORMANCE.md`.
