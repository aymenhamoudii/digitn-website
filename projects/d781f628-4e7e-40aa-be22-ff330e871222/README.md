# Scientific Calculator

A modern, fully functional scientific calculator built with HTML, CSS, and JavaScript. Features include calculation history, memory functions, dark mode, and a comprehensive set of scientific operations.

## Features

### Basic Operations
- Addition, subtraction, multiplication, division
- Parentheses support for complex expressions
- Clear (CE) and All Clear (AC) functions
- Backspace to delete last character

### Scientific Functions
- **Trigonometry**: sin, cos, tan (in degrees)
- **Logarithms**: log (base 10), ln (natural log)
- **Powers & Roots**: Square (x²), Square root (√)
- **Constants**: π (Pi), e (Euler's number)
- **Factorial**: n!

### Memory Functions
- **MC** - Memory Clear
- **MR** - Memory Recall
- **M+** - Add to memory

### Additional Features
- **Calculation History** - View and reuse previous calculations
- **Dark Mode** - Toggle between light and dark themes
- **Persistent Storage** - History and theme preferences saved locally

## Getting Started

1. Open `index.html` in any modern web browser
2. Click buttons to perform calculations
3. Press `=` or Enter to calculate result

### Quick Reference

| Button | Function |
|--------|----------|
| AC | All Clear - Reset calculator |
| CE | Clear Entry - Clear current number |
| √ | Square root |
| x² | Square number |
| sin/cos/tan | Trigonometric functions (degrees) |
| log | Base-10 logarithm |
| ln | Natural logarithm (base e) |
| π | Insert Pi (3.14159...) |
| e | Insert Euler's number (2.71828...) |
| n! | Factorial |
