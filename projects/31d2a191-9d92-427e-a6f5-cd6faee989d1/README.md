# LUMIÈRE | Fine Dining & Fresh Flavors

A production-ready elegant fine-dining restaurant landing page built with a "Bright-fresh" palette and modern gastonomy aesthetic.

## Features
- **Sophisticated Design**: Utilizes serif typography (Cormorant Garamond) and a clean, high-contrast color palette.
- **Dynamic Menu**: Categorized menu with realistic fine-dining item descriptions and price leaders.
- **Online Ordering System**: A fully functional side-drawer ordering interface that manages items and calculates totals.
- **Asymmetric Photo Gallery**: A responsive grid-based gallery showcasing atmospheric visuals and food photography.
- **Location & Reservations**: Integrated Google Maps and structured contact information.
- **Responsive & Mobile-First**: Optimized for all screen sizes from mobile to wide desktops.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, CSS Grid, Flexbox, and fluid typography.
- **JavaScript (Vanilla)**: For menu filtering, order state management, and interaction.
- **Google Fonts**: Cormorant Garamond & Montserrat.

## Setup Instructions
1. Clone or download the project files.
2. Open `index.html` in any modern web browser.
3. No build step or local server is required as the project is designed for `file://` compatibility (no ES modules).

## Design Elements
- **Color Palette**: Pearl neutrals, sage green accents, and slate charcoal text.
- **Typography**: Serif for headings, Sans-serif for body text.
- **Interactions**: Smooth scroll, hover effects, and slide-out navigation.
